# Screenshots de la Documentation

Ce dossier contient les captures d'écran utilisées dans la documentation du projet.

## Coverage Report

Pour générer et capturer le rapport de couverture :

1. Exécuter les tests avec coverage :

   ```bash
   coverage run manage.py test future_skills
   coverage html
   ```

2. Ouvrir le rapport HTML :

   ```bash
   open htmlcov/index.html
   ```

3. Prendre une capture d'écran du rapport et la sauvegarder ici sous le nom :
   - `coverage_module3.png` - Vue d'ensemble de la couverture
   - `coverage_module3_details.png` - Détails par fichier (optionnel)
